

This proyect has the objetive to be a little base to create API REST with NodeJs with Loopback V3
and to be the proyect to get skills with git.



This is the route that I followed:
- git init
- npm init
- (Create .gitignore)
- npm install loopback-cli
- [npx] lb -v
- [npx] lb

- cd (folderName)
-- node .
-- lb datasource
-- lb model
-- lb relation

# Verion 4

- npm install -g @loopback/cli
- lb4 (app_name)
- cd (app_name)
- lb4 datasource
- lb4 model
- lb4 repository
- lb4 controller
- npm start
